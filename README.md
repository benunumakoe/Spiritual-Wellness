# Sanctuary - Catholic Spiritual Wellness App

A mobile-friendly web app offering:
- 🙏 Daily scripture verses based on your emotional state
- 💨 Interactive breathing exercises for stress relief
- 📱 Lock screen widget preview
- 🕯️ Personalized Catholic prayers

## Features
- Choose your mood (Anxious, Lonely, Grateful, Weary, Peaceful)
- Receive matching Bible verses and affirmations
- Guided breath prayer meditation
- Interactive prayer selections
